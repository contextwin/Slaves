rpg制作スタッフ募集要項(趣味)

2020年02月03日

☆現在マップ、フィールド、ダンジョンなどのデザインをお手伝いしていただける絵師さん、グラフィッカーの方急募中です！！！
☆現在、SE、BGMなどの作成を手がけでくださる方急募中です！！！

趣味のプロジェクトですので報酬は用意できないですが、お金儲けはするつもりないですけど、(現在共同開発中のゲーム、Slavesについては)著作権をつけないので、このプロジェクトに参加した経験や、成果物に関しては自由にしてくださって結構です。(肖像権とかに関しては流石に配慮して下さい)
今回のプロジェクト参画することで金銭以外の何かしらメリットがあると思われる方は是非参加して下さい。

現在亡びの風プロジェクトは(C プログラマー 1名, Java プログラマー 1名, イラストレーター 2名)で活動中です。

☆現在開発中のゲーム(Slaves)の開発途中画面です。

-- あると望ましいスキル --
1,日本語が使える。(もしくは今現在学習中)
   年齢不問、学歴不問、性別不問。
   小学校一年生以上でソフトウェア開発自体やソフトウェアを活用した芸術活動,物語作りなどに明るいようならぜひ参加していただきたいです。DM下さい。

技術サイドは、
Linux + C + Xlib + LibSDL2を使用します。
(因みに上記の要件はLinuxのGUIをまずターゲットにしているという意味であり、
あなたのPCがwindowsでもmacでもbsdでも何でもいいです。)

プログラミングに興味あるけど未経験とかでも全然いいです。
シナリオや運用方針に対する疑問質問だけでもありがたいです。

-- 望ましい人物像 --
※グラフィックイメージは担当者が現在2名いる状態です。
ゲーム作ってみたい人。
すでに僕の方でも作りたいゲームの構想が存在しますが、あなたにアイデアがあるならそれのソフト化のプロジェクトを作ってもいいです。わたしもプログラマーとして協力します。シナリオしか無い、とかでもいいです。


-- 方針 --
僕自信が制作しようとしているソフトについては下記のリンク先に原案が書いてます。
http://contextwin.livedoor.blog/archives/5307375.html
(プロジェクト名 = 亡びの風, 制作予定製品名 = Slaves)です。
既に活動方針は僕(contextwin)が何個か用意してますが、批判や不足の指摘は受け付けます。
ただし、批判と指摘を反映するかはこちらでも吟味し、話し合いの上で決定しましょう。
ですので何の作業をしてもらうかは、基本的にちゃんと話し合って、
お互いに納得できる形で進めましょう。
質問が多いほうが望ましいです。
アイデアが多いほうが望ましいです。
結果として質問が少なくなるのは喜ばしいようですが、
わからなくなったら、また質問したり話し合ったりするのが望ましいです。
作業中に詰まったら相談して下さい。
企業への納品などもないので納期もないですからマイペースに作業して下さい。
その上で望ましい双方合意の今回のプロジェクトからの離脱などもあると思います。
その時は僕の方でもプログラマとして手伝えそうなことがあったら参加させて下さい。
そんな感じです。
DMよろしくお願いします。
