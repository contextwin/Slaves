Slaves シナリオ

2020年01月26日

_シナリオや設定の変更について_
活動方針の復唱になりますが、Slavesの発案者はcontextwinだが、だから権威というものでは無いです。
このドキュメントの内容を亡びの風のメンバーは誰でも自由に批判して良いです。
Slavesは原案の初期の段階ではチィポとナイトというキャラクターは性別が未定でしたが、絵師さんとの稟議によってチィポ(女),ナイト(男)というように更新されました。良いアイデアだというふうに意見が噛み合えばシナリオや設定は見直され更新されます。

モンスター類に関して
呪いの部分のボスキャラなどは人＋動物、虫、魚類などの合成した生物
地球に関しては各国の民話や神話を参考にする。例えば日本であれば妖怪、おばけなど。
オリジナルモンスター類に関してはどこに配置するかはモンスターの作者とアイデアを合わせる。

フィールド２、地球

登場するモブキャラクターは各国の民族衣装などで登場する。

旅の流れ
まず呪いの部分に関しては地球よりも都市的であり、地球とは別世界か、または世界地図に架空の大陸として存在するかのどちらかにしようかと考えている。
物語の序盤、主人公たちは呪いの部分以外の国の存在を知らないが。新・光と闇の部分の上位の奴隷たちが地球と取引を行っている。大体の場合不平等な条件での取引。
序盤のシナリオが終わってからは地球を旅するがそこからフリーシナリオっぽくなっていく。
物語が進行すると呪いの部分からの妨害で、貨幣を稼ぐのが難しくなる。
基本的に貨幣は仕事をすることで稼ぐ、モンスタードロップではない。
アラスカから光の部分を攻略して。バビロニアから再び呪いの部分に向かいラストステージ（最も呪われた部分)。
ラストステージはパノプチコンて名前でもいいかも。#!ポリゴポリス、ポリゴニアン、発光少女

_あらすじ＿
この世界には奴隷たちが住んでいた。
最も愚かな奴隷チィポが呪いの部分に生まれた時、大人の奴隷たちはモノを作ってはいけなかった。
モノを作らなければ幸福になれないと考えた光の部分の女神であるオーロラは時々大人の奴隷の身体に取り付いて、子供の奴隷たちに創造を与えようとした。

闇の部分の女神であるヘルは奴隷たちの中でも、労働をしている奴隷の中で、特に労働の才能がありながら呪いの部分で不幸な境遇にあった奴隷たちを闇の部分に住まわせた。
闇の部分の奴隷たちは皆、創作活動をしてはならないと思い込んでいたし、闇の部分の奴隷たち以外の奴隷たちを憎んでいた。が、反面仲間同士ではとても仲がよく全員ヘルだけを唯一の神として信仰している。

元々その昔、神々の間で創作活動や生産のための競争が発生した時、ヘルと闇の部分の奴隷達が一番になった。しかし闇の部分の奴隷たちは闇の部分の奴隷以外を嫌ったので棲み分けられることになり、それ以外の部分をオーロラが統治している。ヘルとオーロラが不仲なわけではない。

主人公たちは呪いの部分の問題を解決するためにオーロラが住むと言われる光の部分（アラスカ）を目指して地球を旅する。(呪いの部分から有機自然的世界観への開放へ向かう軌道)

オーロラに会うのはメインイベントであり、ヘルに会うのはフリーシナリオの方で行かなくてもゲーム自体はクリアできるようにしようか検討中。（隠し要素的なダンジョン、または町）。
！！このシナリオは変更された、光の部分でオーロラと合う前にオーロラの守護としてヘルとボスバトルが発生することにする。ヘルとのバトルはメインストーリーにする。

呪われた部分に生まれたこと自体を憎んでいて地球が嫌い。
呪いの部分に住む奴隷は、だれも完全に幸せであってはならない。
新・光の部分の奴隷はどうすれば自分以外の奴隷に自分たちのために芸術をさせることができるか考えている。
新・闇の部分の奴隷はどうすれば自分以外の奴隷に自分たちのための労働をさせることができるか考えている。

  ヘルの技自由時間の刑について
      ヘルと闇の部分の奴隷が仕事で成果をあげると、呪われた部分の奴隷たちは自由時間を得ることに  なる、チィポなどの味方の奴隷にとってこれは幸福であるが、敵にとっては刑罰となる。
ヘルの登場シーンについて、まず闇の奴隷の中で特に優秀な者たちが鉄格子のオリで囲まれた領域で生活、労働しており、チィポたちがそのマップにはいると鉄格子のオリの上にヘルが立っている。ヘルが嫌いなタイプの奴隷がそこに近づこうとすると精神を崩壊させられる。バトルイベントにするかどうかは未定。普段その鉄格子の上でヘルが鉄格子の中の奴隷を守護している。
　串を刺す
　嫌いな奴隷にここに入ればバレずに横暴に生きることができると教えてオーロラの方を向かせる、オーロラが芸術やオモチャで奴隷を遊ばせていると、ヘルが闇の部分からその奴隷を串刺しにする。
    容姿について
     田植え職用の長靴（ロング）を履いている、膝をついても良いように正面は長靴で膝上まで守られており、後ろは膝裏までのジッパーか何か（か靴下みたいにはく感じ）で防水加工。装飾、配色はキャラデザ担当にお任せします。
  　(アルカロイド)
    黄金宮殿、グロッタの地下

_ゲーム性について_
序盤は武器防具やアイテムなどで自身のスキル不足を補いながら戦うが、
中盤意向は呪いの部分からの攻撃により金欠状態に近づき、
高価な武器防具、アイテムを使用することが難しくなっていく。
つまり序盤のほうが金銭的には裕福な状態に近づけるのが容易である。
キャラクターは、バトルスキルとクリエイティブスキルを成長させることができる。
クリエイティブスキルを使用することでアイテムを消費して新たなアイテムを生成する、
ゲームの後半はクリエイティブスキルで金欠をしのぎながらゲームを進行していくスタイルになっていく。
バトルシステムについて
儀式
 戦闘時、仲間が全員生きている時にステータスアップなどの強化。（オーロラの儀式)
 戦闘時、自分以外の仲間が戦闘不能になるとステータスアップの強化。（自分以外の仲間の自分より強いステータスを自分のものにする、属性優位、アビリティなど）要は一人きりになると他者の長所を模倣する。(ヘルの儀式) { チィポ:耳が無くなる, カノン(攻撃),
　　　　　　　　　　　  ナルシス : 首から上が無くなり攻撃モーションで首から上に三角形に目玉マーク出現。知力が敵と味方の知力の総和以上になり敵の知力が０になる、敵が自分より知力が高かった敵（敵にとっては味方)を攻撃し始める。
                                          シン : 手が無くなる,　エイジングオートメーション、毎ターン敵のステータスがダウンする。
                                          プリンセス : 192連続魔法攻撃,
　　　　　　　　　　　  ギャンブラー: 雷,
                                        }
の２つ。後方は序盤は使用できない。
